Operators in C++

Operators in C++ Operators are used for producing output by performing various types of calculations on two or more inputs. In this lecture, we will see the operators in C++.


#include
using namespace std;

int main(){
    int a=4, b=5;
    cout<<"Operators in C++:"<< endl;
    cout<<"The value of a + b is "<< a+b<< endl;
    cout<< "The value of a - b is "<< a-b << endl;
    cout<< "The value of a * b is "<< a*b<< endl;
    cout<< "The value of a / b is "<< a/b<< endl;
    cout<< "The value of a % b is "<< a%b<< endl;
    cout<< "The value of a++ is "<< a++<< endl;
    cout<< "The value of a-- is "<< a--<< endl;
    cout<< "The value of ++a is "<< ++a<< endl;
    cout<< "The value of --a is "<<--a<< endl;
    cout<< endl;
    return 0;
}

Switch Case Statements in C++

Switch Case Statements in C++ In switch-case statements, the value of the variable is tested with all the cases. An example program for the switch case statement.

#include
using namespace std;

int main(){
    int age;
    cout<< "Tell me your age"<< endl;
        cin>>age;
    switch (age)
    {
    case 18: 
        cout<<"You are 18"<< endl;
        break;
    case 22: 
        cout<< "You are 22"<< endl;
        break;
    case 2: 
        cout<< "You are 2"<< endl;
        break;
    default:
    cout<< "No special cases"<< endl;
        break;
    }
    cout<< "Done with switch case";
    return 0;
    }
