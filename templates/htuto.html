HTML Headings

HTML defines six levels of headings. A heading element implies all the font changes, paragraph breaks before and after, and any white space necessary to render the heading. The heading elements are H1, H2, H3, H4, H5, and H6 with H1 being the highest (or most important) level and H6 the leas

<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document</title>
    </head>
    <body>
        <h1>This is heading 1</h1>
        <h2>This is heading 2</h2>
        <h3>This is heading 3</h3>
        <h4>This is heading 4</h4>
        <h5>This is heading 5</h5>
        <h6>This is heading 6</h6>                           
    </body>
</html>

HTML Table

The HTML tables allow web authors to arrange data like text, images, links, other tables, etc. into rows and columns of cells.The HTML tables are created using the table tag in which the tr tag is used to create table rows and td tag is used to create data cells. The elements under td are regular and left aligned by default

<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document</title>
    </head>
    <body>
        <table>
            <tr>
                <th>Heder1</th> <!-- th tag is for Table heading -->
                <th>Heder2</th>
                <th>Heder3</th>
            </tr>
            <tr>
                <td>Row 1 content1</td>
                <td>Row 1 content2</td>
                <td>Row 1 content3</td>
            </tr>
            <tr>
                <td>Row 2 content1</td>
                <td>Row 2 content2</td>
                <td>Row 2 content3</td>
            </tr>
            <tr>
                <td>Row 3 content1</td>
                <td>Row 3 content2</td>
                <td>Row 3 content3</td>
            </tr>
        </table>                       
    </body>
</html>

HTML Form

An HTML form is used to collect user input. The user input is most often sent to a server for processing.

<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document</title>
    </head>
    <body>
        <form action="example.html" method="POST" class="contact-c">
            <input type="text" name="name" id="name" placeholder="Enter Your Name">
            <input type="number" name="phone" id="phone" placeholder="Enter Your Mobile No">
            <input type="email" name="email" id="email" placeholder="Enter Your email">
            <input type="number" name="age" id="age" placeholder="Enter Your age">
            <textarea name="concern" id="" cols="30" rows="10" ></textarea>
            <button class="contact-btn">Submit</button>
        </form>                            
    </body>
</html>

HTML Lists

There are three list types in HTML: unordered list — used to group a set of related items in no particular order. ordered list — used to group a set of related items in a specific order. description list — used to display name/value pairs such as terms and definitions.

<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document</title>
    </head>
    <body>
        <!-- Unordered List -->
        <ul>
            <li>First list item of unordered list </li>
            <li>Second list item of unordered list  </li>
            <li>Third list item of unordered list  </li>
        </ul>

        <!-- ordered List -->
        <ol>
            <li>First list item of ordered list </li>
            <li>Second list item of ordered list  </li>
            <li>Third list item of ordered list  </li>
        </ol>                               
    </body>
</html>
