CSS Tutorial

CSS Basic syntax:

CSS is the language we use to style an HTML document. CSS is the language we use to style an HTML document. CSS describes how HTML elements should be displayed

 
/*Example*/

Selctor{
    property:value;
}

/*class Selector*/

.className{
    property:value;
}

/*ID Selector*/

#idName{
    property:value;
}

Universal And Body Selector

The universal selector (*) selects all HTML elements on the page And body selector changes HTML body style.

 
/*Universal and Body Selector*/

*{
    margin:0;
    padding:0;
}

body{
    Overflow:hidden;
    font-family:monospace;
}

CSS Colors:

Colors are specified using predefined color names, or RGB, HEX, HSL, RGBA, HSLA values. 

 
/*Example*/

h1{
    color:red;
    background-color:black;
}

p{
    color:white;
    background-color:yellow;
}

.class{
    color:#fff; /*Hexa color*/
    background-color:rgb(0,0,0); /*RGB color*/
}

CSS Margins & Padding:

All the margin properties can have the following values: auto - the browser calculates the margin length - specifies a margin in px, pt, cm, etc. % - specifies a margin in % of the width of the containing element inherit - specifies that the margin should be inherited from the parent element 

 
/*Example*/
/* Margin & padding*/
p {
    margin-top: 100px;
    margin-bottom: 100px;
    margin-right: 150px;
    margin-left: 80px;
}
p {
    padding-top: 100px;
    padding-bottom: 100px;
    padding-right: 150px;
    padding-left: 80px;
}

CSS Pseudo-elements

A CSS pseudo-element is used to style specified parts of an element. For example, it can be used to: Style the first letter, or line, of an element Insert content before, or after, the content of an element 

 
/*Example*/
/* Pseudo Selector*/
p::first-line {
    color: #ff0000;
    font-variant: small-caps;
}
.class:before{
    content:"";
    background:url('first.jpg');
    height:100vh;
    width:100%;
}
